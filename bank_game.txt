БАНКОВСКИЙ БИЗНЕС (карточная модель)

Игроков: 2–6
Время: 60–90 мин
Цель: сохранить платёжеспособность и нарастить капитал. Побеждает банк с
максимальным собственным капиталом через 12 раундов (или последний выживший).

═══════════════════════════════════════════════════════════════
КОМПОНЕНТЫ
═══════════════════════════════════════════════════════════════

1. ХРАНИЛИЩЕ — вырубная картонная 3D-модель сейфа (собирается в начале).
   У каждого игрока своё. В нём хранятся наличные деньги банка (касса).

2. ДЕНЬГИ (купюры): 1K, 5K, 10K, 50K, 100K, 500K.

3. КАРТЫ (всего 110 шт.):
   - Акции (20 шт., зелёные) — выпускаются банком для привлечения капитала
   - Векселя (20 шт., синие) — долговые обязательства
   - События (40 шт., красные) — ситуации, влияющие на баланс банка
   - Контрагенты (30 шт., жёлтые) — клиенты, депозиты, кредиты

4. БЛАНК БАЛАНСА — у каждого игрока лист с двумя колонками:

   ┌────────────────────────┬────────────────────────┐
   │ АКТИВЫ                 │ ПАССИВЫ                 │
   ├────────────────────────┼────────────────────────┤
   │ Касса (наличеые)       │ Акционерный капитал    │
   │ Хранилище              │ (оплаченные акции)      │
   │ Выданные кредиты       │ Привлечённые депозиты  │
   │ Приобретённые векселя  │ Выпущенные векселя     │
   │ Ценные бумаги          │ Межбанковские кредиты  │
   │ Корсчёт в ЦБ           │ Резервы                │
   │ Прочие активы          │ Прибыль (убыток)       │
   │                        │ Дивиденды к выплате    │
   │                        │ Резерв под потери      │
   ├────────────────────────┼────────────────────────┤
   │ ИТОГО АКТИВЫ           │ ИТОГО ПАССИВЫ          │
   └────────────────────────┴────────────────────────┘

═══════════════════════════════════════════════════════════════
НАЧАЛО ИГРЫ
═══════════════════════════════════════════════════════════════

1. Каждый игрок собирает своё ХРАНИЛИЩЕ из картонной вырубки.
   Стоимость хранилища = 50K (списывается с акционерного капитала).
   Вариант: хранилище можно не собирать, а положить лист с вырубкой
   перед собой — это символический актив «Хранилище».

2. КАПИТАЛИЗАЦИЯ:
   — Каждый игрок ПОКУПАЕТ АКЦИИ своего банка:
     • Одна акция = 100K (номинал)
     • Можно купить 1–5 акций на старте
     • Деньги за акции = АКЦИОНЕРНЫЙ КАПИТАЛ банка
   — Пример: купил 3 акции × 100K = 300K акционерного капитала

3. ИЗ АКЦИОНЕРНОГО КАПИТАЛА сразу тратится:
   — Хранилище: –50K (становится активом)
   — Обязательный резерв в ЦБ: 10% от капитала (блокируется на корсчёте)
   — Остаток = касса (наличные в хранилище)

4. Каждому игроку выдаётся:
   — 1 карта СОБЫТИЕ (в руку)
   — 1 карта КОНТРАГЕНТ (в руку)

═══════════════════════════════════════════════════════════════
СТРУКТУРА ХОДА
═══════════════════════════════════════════════════════════════

Каждый раунд = все игроки ходят по очереди. В свой ход:

ФАЗА 1 — СОБЫТИЕ (обязательно)
  — Тянете карту СОБЫТИЕ и применяете эффект

ФАЗА 2 — ДЕЙСТВИЯ (до 3 действий, можно повторять)
  Действия:
    • Привлечь депозит (сыграть Контрагента-депозит с руки)
    • Выдать кредит (сыграть Контрагента-кредит с руки)
    • Выпустить вексель (взять из колоды Векселей, заполнить)
    • Купить вексель (у банка или игрока)
    • Продать вексель
    • Выпустить акции (новая эмиссия)
    • Выкупить свои акции (уменьшить капитал)
    • Привлечь межбанковский кредит (из колоды Межбанк)
    • Погасить обязательство досрочно
    • Сделать резерв под потери (отчислить с прибыли)
    • Объявить дивиденды (начислить, выплатить из прибыли)
    • Провести сделку с другим игроком (вексель/кредит/депозит)
    • Продать/купить актив у другого банка

ФАЗА 3 — РАСЧЁТЫ (автоматически, в конце хода)
  — Начисляются % по депозитам (расход)
  — Начисляются % по выданным кредитам (доход)
  — Начисляются % по векселям
  — Начисляются операционные расходы (аренда, инкассация и т.д.)
  — Проверка нормативов: резервы ≥ 10% от депозитов
  — Если капитал (активы − пассивы) отрицательный — банкротство

═ РАЗ В 4 РАУНДА (СБОР ДИВИДЕНДОВ) ═
  — Каждый банк обязан выплатить дивиденды акционерам
  — Размер: 5% от чистой прибыли за 4 раунда (или фикс 10K на акцию)
  — Если прибыли нет — дивиденды не выплачиваются (но репутация падает)

═══════════════════════════════════════════════════════════════
ПРИМЕР БАЛАНСА (СТАРТ)
═══════════════════════════════════════════════════════════════

Игрок купил 3 акции × 100K = 300K капитала

АКТИВЫ:
  Хранилище                50K
  Касса                   220K  (300K − 50K − 30K резерв)
  Корсчёт в ЦБ (резерв)   30K   (10% × 300K)
  ─────────────────────────────
  ИТОГО АКТИВЫ            300K

ПАССИВЫ:
  Акционерный капитал     300K  (3 акции × 100K)
  ─────────────────────────────
  ИТОГО ПАССИВЫ           300K

Баланс сошёлся: 300K = 300K

═══════════════════════════════════════════════════════════════
БАНКРОТСТВО
═══════════════════════════════════════════════════════════════

Если после расчётов капитал (Активы − Пассивы) < 0:
  — Банк считается неплатёжеспособным
  — Игрок может попросить помощь у другого банка (вход в капитал)
  — Если помощи нет — банкротство, игрок выбывает
  — Его активы распродаются: депозиты возвращаются (если хватает),
    кредиты переходят другим банкам (дисконт 50%)
  — Акционеры теряют свои вложения

═══════════════════════════════════════════════════════════════
ПОБЕДА
═══════════════════════════════════════════════════════════════

После 12 раундов каждый игрок считает:
  СОБСТВЕННЫЙ КАПИТАЛ = Активы − Пассивы

Побеждает банк с максимальным капиталом.

Также можно играть до последнего выжившего (единственный банк,
не обанкротившийся) — он побеждает автоматически.

═══════════════════════════════════════════════════════════════
ДОПОЛНИТЕЛЬНЫЕ ПРАВИЛА (ОПЦИОНАЛЬНО)
═══════════════════════════════════════════════════════════════

1. СОБРАНИЕ АКЦИОНЕРОВ — раз в 4 раунда игроки голосуют числом
   акций (обыкновенные, каждая = 1 голос). Решают:
   — менять ли стратегию (снизить/повысить риски)
   — переизбирать ли менеджмент (штраф 20K)
   — одобрить ли допэмиссию акций

2. КРЕДИТНЫЙ РИСК — при выдаче кредита бросьте монетку:
   — Орёл: клиент платит исправно
   — Решка: клиент просрочил (не платит этот раунд)

3. РЫНОЧНАЯ ЦЕНА АКЦИЙ — акции банка могут торговаться выше
   номинала, если банк стабилен. Каждые 4 раунда:
   — Если капитал вырос с прошлого собрания: цена +10%
   — Если упал: цена −10%
   — Если банк платил дивиденды: цена +5%
