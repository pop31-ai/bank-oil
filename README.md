# БАНКЪ & НЕФТЬ: ГОРОДСКАЯ АРХИТЕКТУРА

Карточная CLI-игра о банках, нефти и городских владениях для 2-4 игроков.

## Как запустить

```
python bank_game_city.py
```

Требуется Python 3.10+.

## Правила

Каждый игрок начинает с банком в Финансовом квартале, выпускает акции и покупает районы. Цель — нарастить капитал за 10 раундов.

### Районы города (9 шт.)

| Район | Цена | Бонус |
|-------|------|-------|
| Финансовый квартал | 0K | Стартовый район, хранилище |
| Биржевая площадь | 200K | Эмиссия акций |
| Гостиный двор | 200K | Векселя |
| Мэрия | 200K | Отмена события |
| Порт | 250K | Международные контрагенты |
| Вокзал | 200K | +10K/раунд |
| Промышленная зона | 250K | Нефть +20% |
| Банк России | 300K | Рефинанс |
| Торговый центр | 200K | Депозиты +10% |
| Особняк купца | 500K | +1 действие/ход (секретный) |

### Действия за ход (3 шт.)

1. Сыграть карту из руки (кредит, депозит, нефть, межбанк)
2. Купить район (смежный с вашими)
3. Открыть филиал (100K)
4. Выпустить акции (Биржа)
5. Выпустить вексель (Гостиный двор)
6. Карта города
7. Газета (15 статей)
8. Закончить ход

### События

Каждый ход игрок тянет событие из Мэрии. 24 события: кризисы, инфляция, нефтяные бумы, аудит ЦБ и др.

## AI Герои с прокачкой

При создании игры можно назначить любого игрока роботом. Каждый робот — это герой с уникальными бонусами и системой уровней.

### Герои

| Герой | Титул | Бонус | Описание |
|-------|-------|-------|----------|
| Купец Гольдман | Торговый магнат | Скидка на районы -20K/уровень | Скупает районы со скидкой |
| Кассир Сомов | Хранитель кассы | Депозиты на 1% дешевле/уровень | Консервативный банкир |
| Директор Ковалёва | Железная леди | +1 действие/3 уровня | Агрессивная экспансия |
| Риск-менеджер Лосев | Спасатель капиталов | Нейтрализует 1 плохое событие/раунд (с 2 ур.) | Играет от обороны |

### Прокачка

- **XP** даётся за: выживание (+2/раунд), прибыль (+1/100K), районы (+1/шт.), контракты (+1/2 шт.)
- **Порог уровня**: `lv * (lv+1) * 3` XP (6, 18, 36, 60, 90...)
- **Макс. уровень**: 10

### Бонусы за уровень

| Уровень | Скидка на районы | Доп. действия |
|---------|------------------|---------------|
| 1 | 20K | 0 |
| 2 | 40K | 0 |
| 3 | 60K | +1 |
| 4 | 80K | +1 |
| 5 | 100K | +1 |
| 6 | 100K | +2 |
| 7-8 | 100K | +2 |
| 9+ | 100K | +3 |
